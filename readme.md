# Modeling & Simulation
A excerse for Hogeschool Rotterdam, class Modeling & simulation.

## Running and compilling

Windows, cmake 3.15, visual studio 2022
```sh
cd .\simulation\build\
mkdir build
cd .\build\
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build . --config Debug 
.\Debug\OpenGLSimulation
```
